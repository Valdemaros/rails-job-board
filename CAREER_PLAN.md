## 📅 План на 6 месяцев

### **Месяц 1-2: Фундамент**

**Технологии:**
- ✅ Rails (углублённо)
- ✅ Telegram боты
- ✅ Тесты (Minitest/RSpec)
- 📚 SQL, Active Record глубоко
- 📚 Git (ветки, rebase, PR)
- 📚 Docker (базово)

**Проект 1:** Telegram бот с подписками на вакансии (этот проект)

**Критерии завершения:**
- Бот работает в продакшене
- Есть подписки, уведомления, фильтры
- Покрыто тестами (80%+)
- README с документацией
- Выложено на GitHub

---

### **Месяц 3-4: Портфолио**

**Технологии:**
- 📚 API (JSON, REST, GraphQL)
- 📚 Frontend (Stimulus, Turbo, JavaScript)
- 📚 Деплой (Kamal, Docker, VPS)
- 📚 Sidekiq, Redis, кэширование

**Проект 2:** Fullstack приложение с командной работой
- Например: трекер задач с реальным временем
- Или: маркетплейс с корзиной и заказами

**Проект 3:** Интеграция с внешними API
- Платежи (Stripe, PayPal)
- Карты (Google Maps, OpenStreetMap)
- Соцсети (OAuth, авторизация)

**Критерии завершения:**
- 2-3 рабочих проекта на GitHub
- Деплой на реальный сервер
- Тесты, CI/CD
- Документация

---

### **Месяц 5: Углубление**

**Технологии:**
- 📚 Архитектура (Service Objects, Patterns)
- 📚 Производительность (N+1, индексы, кэширование)
- 📚 Безопасность (XSS, CSRF, SQL injection)
- 📚 Code review (открыть PR в чужие проекты)

**Проект 4:** SaaS с подпиской
- Платные тарифы
- Email рассылки
- Админка
- Аналитика

**Критерии завершения:**
- Рабочий SaaS с реальными пользователями
- Платежи работают
- Логи, мониторинг

---

### **Месяц 6: Поиск работы**

**Подготовка:**
- 📝 Резюме (выделить проекты, не инженера)
- 📝 GitHub (причёсать, README, тесты)
- 📝 LinkedIn (заполнить, рекрутеры)
- 📝 Собесы (алгоритмы, SQL, Rails)

**Активности:**
- Отклики на вакансии (10-20 в неделю)
- Нетворкинг (митапы, конференции)
- Пробные собеседования

**Критерии завершения:**
- 50+ откликов
- 10+ собеседований
- 1+ оффер

## ⚡ План на 1 неделю (Telegram бот)

### **День 1-2: Модель и подписка**

```
- Миграция для Subscription (telegram_id, language, active)
- Валидации в модели
- Команда /subscribe (просто текстом)
- Команда /status
```

**Чеклист:**
- [ ] Миграция создана
- [ ] Модель с валидациями
- [ ] /subscribe работает
- [ ] /status показывает подписку
- [ ] Тесты на модель

---

### **День 3-4: Уведомления**

```
- TelegramNotifier (сервис отправки)
- Изменить FetchVacanciesJob (отправка после парсинга)
- Фильтр по language
```

**Чеклист:**
- [ ] TelegramNotifier создан
- [ ] FetchVacanciesJob обновлён
- [ ] VacancyFilter фильтрует по языку
- [ ] Уведомления приходят в Telegram
- [ ] Тесты на сервис

---

### **День 5: Фильтры**

```
- Вакансии не старше 2 недель
- Команда /unsubscribe
```

**Чеклист:**
- [ ] Фильтр по дате публикации
- [ ] /unsubscribe работает
- [ ] Тесты на фильтры

---

### **День 6: Тесты**

```
- Тесты на Subscription модель
- Тесты на TelegramNotifier
- Integration тесты
```

**Чеклист:**
- [ ] 80%+ покрытие
- [ ] Все тесты зелёные
- [ ] CI настроен (опционально)

---

### **День 7: Полировка**

```
- Баги, логирование
- Деплой на сервер (если есть)
- README в GitHub
```

**Чеклист:**
- [ ] Баги исправлены

## 💬 Заметки

**Важно:**
- Не торопись (лучше 6 месяцев качественно чем 3 наспех)
- Делай проекты которые **заканчиваешь** (не 10 начатых)
- Ищи ментора (кто-то кто уже работает)
- Начни откликаться через 4 месяца (пробовать собесы)

**Ежедневная рутина:**
- 4-6 часов кодинга
- 1 час английского
- 30 минут чтение (книги/блоги)
- Перерывы каждые 45 минут

**Отслеживать:**
- GitHub contributions (зелёные квадраты)
- Изученные технологии (список)
- Прочитанные книги
- Написанные статьи

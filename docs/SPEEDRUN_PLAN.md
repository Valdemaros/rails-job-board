### Почему Solid Queue вместо Sidekiq?

| Критерий | Sidekiq | Solid Queue |
|----------|---------|-------------|
| Требует Redis | ✅ Да | ❌ Нет (только PostgreSQL) |
| Отдельный воркер | ✅ Нужен | ❌ Встроен в Rails 8 |
| Бесплатно на Render | ❌ $7/мес | ✅ $0 |
| Cron планировщик | ✅ sidekiq-cron | ✅ Встроенный |
| Сложность настройки | Средняя | Низкая |

## 🗄️ Схема Базы Данных

### Таблица: `subscriptions`

| Колонка | Тип | Nullable | Default | Описание |
|---------|-----|----------|---------|----------|
| `id` | bigint | ❌ NO | auto | Первичный ключ |
| `telegram_id` | bigint | ❌ NO | — | Уникальный ID пользователя Telegram |
| `username` | string | ✅ YES | NULL | Username пользователя |
| `language` | string | ❌ NO | — | Язык (ruby, python, java, go, javascript, php) |
| `active` | boolean | ✅ YES | true | Активна ли подписка |
| `created_at` | datetime | ❌ NO | — | Когда подписался |
| `updated_at` | datetime | ❌ NO | — | Когда обновил подписку |

**Индексы:**
```sql
CREATE UNIQUE INDEX index_subscriptions_on_telegram_id ON subscriptions(telegram_id);
CREATE INDEX index_subscriptions_on_language ON subscriptions(language);
CREATE INDEX index_subscriptions_on_active ON subscriptions(active);
```

## 🔧 Технические детали

### Solid Queue Cron

```yaml
# config/solid_queue.yml
default: &default
  dispatchers:
    - polling_interval: 1
      batch_size: 500
  workers:
    - queues: default
      threads: 3
      processes: 1
      polling_interval: 0.1

development:
  <<: *default

production:
  <<: *default
```

### Планировщик задач

```ruby
# config/schedule.rb (или через Solid Queue built-in)
SolidQueue::Job.create!(
  class_name: "FetchVacanciesJob",
  cron: "*/5 * * * *",  # Каждые 5 минут
  queue_name: "default"
)
```

## ⚠️ Риски и решения

| Риск | Решение |
|------|---------|
| Render засыпает (free tier) | Ping каждые 5 мин (UptimeRobot) |
| HH.ru блокирует API | User-Agent, задержки между запросами |
| Telegram webhook не работает | Проверить ngrok, SSL сертификат |
| БД переполняется | Удалять вакансии старше 30 дней |
| Рассылка спам | Не чаще 1 раза в 5 минут, кнопка отписки |
